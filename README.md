# cmpe210-ControlStaturationProject

## Set Up
The topology consists of 3 hosts (2 clients, 1 attacker) and 1 Open Flow 1.3 switch.
Run the following commands in your terminal
```
$ chmod +x topo_setup.sh
$ ./topo_setup.sh
```

If you already installed everything and just want to run ryu and mininet, run:
```
$ USE_APT_INSTALL=0 ./topo_setup.sh
```
